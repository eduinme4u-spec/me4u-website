# ME4U Academy Full-Stack Roadmap

## Target Product

ME4U Academy should become both a public business website and an internal lead-management web app.

## Version 1

- Public website
- Course sections
- Lead generation form
- Supabase database
- Admin login page
- Admin dashboard
- Lead list
- Lead status tracking
- Follow-up date tracking

## Version 2

- Edit lead details
- Add follow-up notes
- Search and filters
- Export leads to CSV
- WhatsApp follow-up links
- Admin user roles

## Version 3

- Student login
- Batch management
- Attendance
- Fee tracking
- Certificates
- Payment integration

## Recommended Deployment

- Code: GitHub
- App hosting: Vercel
- Database/Auth: Supabase
- Domain: `me4u.co.in` or a subdomain such as `academy.me4u.co.in`
